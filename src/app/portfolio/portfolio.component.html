Portfolio Project

Finance Application

A modern finance app built with .NET (backend) and Angular (frontend), deployed via GitHub Actions to Azure Container Apps.

Key Features:

  • User registration with email confirmation, password reset and token-based authentication using HTTP Only cookies
  • CRUD operations for transactions and transaction groups
  • CSV import for automatic transaction creation
  • LLM integration for smart transaction categorization and queries
  • Transaction filtering, sorting, and currency switching
  • Weekly background job to fetch exchange rates from external APIs
